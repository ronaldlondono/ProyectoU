Registro de Usuario

¡Tu bienestar mental es nuestra prioridad! ¡Da el primer paso hacia una mente más saludable y equilibrada! Inscríbete en nuestra asesoría psicológica hoy y comienza a construir un camino hacia el bienestar emocional y la realización personal. Estamos aquí para apoyarte en cada paso del camino.

¡Prepárate para desatar tu máximo potencial y conquistar el mundo laboral! ¡Tu potencial es infinito; hoy es el día para desatarlo! La vida deportiva es una carrera emocionante y desafiante, y estamos aquí para potenciar tu talento y determinación. No solo te preparamos para competir, sino para liderar, inspirar y alcanzar la grandeza dentro y fuera del campo de juego.

Información del aplicativo

¿Cómo iniciar sesión?

Después de completar el registro, recibirás información para acceder al inicio de sesión.
Una vez que tengas las credenciales, ingresa el nombre de usuario y la contraseña proporcionados.

Paso siguiente: agendamiento

¿Cómo agendar una asesoria?

  1. Codigo estudiante: Introduce el código que se le asignó al estudiante.
  2. Codigo Adm: Tu código de administrador.
  3. Codigo Servicio:
    Codigo "1":asesoria Psicologica.
    Codigo "2": asesoria para la vida laboral.
    Codigo "3": asesoria para la vida deportiva.
  4. Hora inicial, final y día: En qué se hará la tutoría.
  5. Motivo:el porqué el estudiante requiere la asesoría.

Este paso es solo para Administradores.
